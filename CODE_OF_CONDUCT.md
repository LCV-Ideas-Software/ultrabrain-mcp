# Code of Conduct

This project adopts the **Contributor Covenant 3.0** as its community standard.
The canonical text is available at:

- https://www.contributor-covenant.org/version/3/0/code_of_conduct/
- Plain-text mirror: https://www.contributor-covenant.org/version/3/0/code_of_conduct/code_of_conduct.md

By participating in this project, you agree to follow that standard.

## Scope

This applies to all project spaces on GitHub, including issues, pull requests,
discussions, commits, and public representation of the project.

## Enforcement Contact

Reports can be sent privately to:

**lcv@lcv.dev**

Reports will be acknowledged within 48 hours and handled according to the
Contributor Covenant 3.0 enforcement ladder. Reporter identity will be kept
confidential.
